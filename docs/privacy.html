Política de Privacidade – Glico+

Última atualização: 24/02/2026
Resumo:
Os registros de glicemia ficam armazenados localmente no seu dispositivo. Para melhorar estabilidade e entender uso do app, coletamos dados técnicos e anônimos via Firebase (Analytics e Crashlytics).

1. Dados armazenados localmente

Os dados inseridos pelo usuário no Glico+ (como registros e informações relacionadas) permanecem armazenados no dispositivo.

2. Dados técnicos coletados

O Glico+ utiliza serviços do Google Firebase para coletar dados técnicos e anônimos, com as seguintes finalidades:

  • Firebase Analytics: entender, de forma agregada, como o app é usado (ex.: telas acessadas, ações realizadas).
  • Firebase Crashlytics: identificar falhas e crashes para melhorar a estabilidade.

Esses dados podem incluir informações como modelo do dispositivo, versão do sistema operacional, eventos de navegação no app e relatórios de erro.

Importante: não coletamos nem enviamos para o Firebase valores individuais de glicemia como parte da telemetria.

3. Compartilhamento

O Glico+ não vende dados pessoais. O compartilhamento ocorre apenas na medida necessária para o funcionamento dos serviços mencionados (Firebase Analytics e Crashlytics), com finalidade de melhoria do aplicativo.

4. Exclusão de dados

O usuário pode excluir os dados armazenados localmente pelo aplicativo usando a opção de exclusão disponível no próprio app.

5. Alterações desta política

Esta política pode ser atualizada para refletir melhorias do aplicativo. A data de “Última atualização” será ajustada quando isso ocorrer.
