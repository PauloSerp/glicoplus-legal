Política de Privacidade – Glico+

Última atualização: 24/02/2026
Resumo:
Os registros de glicemia ficam armazenados localmente no seu dispositivo. Para melhorar estabilidade e entender uso do app, coletamos dados técnicos e anônimos via Firebase (Analytics e Crashlytics).

1. Dados armazenados pelo aplicativo

O Glico+ permite que o usuário registre informações relacionadas à sua saúde, como medições de glicemia, refeições e dados de acompanhamento. Esses dados podem ser armazenados:

  • Localmente no dispositivo do usuário;
  • Na nuvem, caso o usuário opte por utilizar a funcionalidade de sincronização de conta.

2. Sincronização e armazenamento em nuvem

Quando o usuário ativa a sincronização de conta, os dados registrados no aplicativo passam a ser armazenados de forma segura em serviços do Google Firebase, incluindo:

  • Firebase Authentication (para gerenciamento de conta);
  • Cloud Firestore (para armazenamento dos dados do usuário).

Essa sincronização permite backup e acesso aos dados em diferentes dispositivos.

3. Dados técnicos coletados

O Glico+ utiliza serviços do Google Firebase para coletar dados técnicos e anônimos, com as seguintes finalidades:

  • Firebase Analytics: entender, de forma agregada, como o app é utilizado;
  • Firebase Crashlytics: identificar falhas e melhorar a estabilidade.

Esses dados podem incluir informações como modelo do dispositivo, versão do sistema operacional e eventos de uso do aplicativo.

4. Dados de saúde

As informações registradas pelo usuário no Glico+ podem incluir dados relacionados à saúde. Esses dados são fornecidos diretamente pelo usuário e são utilizados exclusivamente para exibição, organização e funcionalidades do aplicativo.

O Glico+ não realiza diagnóstico médico e não substitui acompanhamento profissional.

5. Compartilhamento de dados

O Glico+ não vende dados pessoais. Os dados podem ser processados por serviços do Google Firebase exclusivamente para viabilizar funcionalidades como autenticação, sincronização e estabilidade do aplicativo.

6. Exclusão de dados

O usuário pode:

  • Excluir registros diretamente pelo aplicativo;
  • Interromper a sincronização de dados a qualquer momento;
  • Solicitar a exclusão completa dos dados entrando em contato pelo e-mail informado abaixo.

7. Segurança

O Glico+ utiliza mecanismos de autenticação e regras de acesso para proteger os dados do usuário. As informações são armazenadas em infraestrutura segura fornecida pelo Google Firebase.

8. Alterações desta política

Esta política pode ser atualizada para refletir melhorias e novas funcionalidades do aplicativo.

9. Contato
